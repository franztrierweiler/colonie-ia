# Colonie-IA
Un clone de jeu galactique de cowboys des années 1990.

Créé par Franz Trierweiler en décembre 2025 avec Claude Code, dans l'esprit d'Hebdogiciel (Zlika, zlika, bidouille grenouille et le Banni)

J'ai créé ce projet afin de comprendre les tenants et aboutissants du SDD (Spec Driven Development). J'ai essayé d'utiliser progressivement toutes les fonctions de Claude Code, notamment la planification de tâches, la discussion d'architectures techniques avec l'IA, le repect de règles de sécurité, la création de tests et le déploiement en continu.
